# 🤖 UniHelp AI Chatbot (Django + DRF + Gemini AI)

This is an AI-powered chatbot backend built with **Django REST Framework** and **Google Gemini API**.  
It is designed to power a smart university assistant that can answer student queries using AI.

---

## 🚀 Features

- Django REST Framework API
- Google Gemini AI integration
- Custom system prompt support
- JSON-based chatbot response
- CORS enabled for React frontend
- Secure API key management using `.env`

---

## 🧠 Project Purpose

UniHelp is an AI assistant that can help students with:

- Academic questions
- Course-related information
- General university support
- Smart AI-based responses using Gemini

---

## 🛠️ Tech Stack

- Python 3.11+
- Django 5+
- Django REST Framework
- Google Generative AI (Gemini 2.5 Flash)
- python-dotenv
- django-cors-headers
